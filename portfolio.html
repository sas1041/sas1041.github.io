Description:

This is a school management software created for my Computer Science capstone project. It was built using a variety of backend and frontend languages and has a persistent MySQL database running on a Tomcat server. The purpose of this software was to manage a school districts' data and then create school bus routes for the student's within the district. The software features role based authentication with four different roles with different permissions. Utilizes Google Maps API to determine locational data and determine efficient bus routes. It also features uploadig/exporting data using CSVs and has a database backup and restore feature. setup.

Languages Used:

  • Java
  • SQL
  • Javascript
  • HTML
  • CSS

Resources Used:

Backend

  • Spring Boot/Thymeleaf
  • JUnit
  • Java Mail Sender
  • Google Maps API
  • MySQL Server & Database

Frontend

  • DataTables

  • Bootstrap
